# Insurance Premium Prediction

Jupyter notebook: `insurance_premium_prediction.ipynb`

Description:
- Tabular regression model to predict insurance premiums (data cleaning, feature engineering, model training).

How to run:
1. Open the notebook in Jupyter or JupyterLab.
2. Install dependencies (example):

```
pip install scikit-learn pandas numpy matplotlib
```

Notes:
- This folder is prepared to become a separate repository. Use `scripts/package_repos.py` to copy the notebook, initialize a git repo, and optionally push to GitHub.
